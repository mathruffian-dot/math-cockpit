📐重點 1：三角形內角和為 180°

📖 簡報講義

🧪 互動探索：拖動頂點觀察角度

∠A + ∠B + ∠C = 180°

拖動三角形的頂點，觀察三個角的和永遠是 180°

📝 形成性評量：△ABC 中，若 ∠A = 45°、∠B = 70°，則 ∠C = ？
∠C = 180° − 45° − 70° = 65°。三角形內角和為 180°，所以第三個角用 180° 減去已知的兩角即可。

🖼️ 資訊圖表

💡 教學提示

  • 可搭配附件摺紙活動，讓學生將三角形三個角摺到同一直線上
  • 強調：內角和的成立與邊長無關，僅與角度有關
  • 常見錯誤：學生容易忘記「180° 減兩角」的步驟
📐重點 2：三角形外角定理

📖 簡報講義

🧪 互動探索：外角 = 不相鄰兩內角之和

外角 = ∠A + ∠B

拖動頂點觀察外角與不相鄰內角的關係

📝 形成性評量：△ABC 中，∠B 的外角為 120°，且 ∠C = 2∠A，則 ∠A = ？
由外角定理：∠A + ∠C = 120°。因為 ∠C = 2∠A，所以 ∠A + 2∠A = 120°，3∠A = 120°，∠A = 40°。

🖼️ 資訊圖表

💡 教學提示

  • 外角定理是解題捷徑：不需先算第三個內角，直接用兩個不相鄰內角的和
  • 用繞行三角形公園的情境引入「外角」概念最直觀
  • 常見錯誤：把「相鄰內角」和「不相鄰內角」搞混
📐重點 3：多邊形內角和公式

📖 簡報講義

🧪 互動探索：n 邊形內角和

5
內角和 = (5−2) × 180° = 540°
📝 形成性評量：八邊形的內角和為多少度？
八邊形內角和 = (8−2) × 180° = 6 × 180° = 1080°。由一個頂點畫對角線可分割成 6 個三角形。

🖼️ 資訊圖表

💡 教學提示

  • 引導學生「由一個頂點畫對角線」的探索方式來推導公式
  • 可用表格整理 n=3~7 的對角線數、三角形數、內角和
  • 注意動動腦：從內部一點分割的三角形數 ≠ (n-2)
📐重點 4：外角和恆為 360°

📖 簡報講義

🧪 互動探索：外角和 = 360°

5
n 邊形的一組外角和 = 360°
📝 形成性評量：正五邊形的一個外角為多少度？
正五邊形的一個外角 = 360° ÷ 5 = 72°。外角和恆為 360°，正多邊形每個外角相等，所以除以邊數即可。

📐重點 5：凸多邊形與凹多邊形

📖 簡報講義

🧪 互動探索：凸 vs 凹

凸多邊形：所有內角 < 180°
📝 形成性評量：下列關於凹多邊形的敘述，何者正確？
凹多邊形至少有一個內角大於 180°（又稱凹角或優角），且至少有一條對角線會在多邊形外部。

📐重點 6：正多邊形的單一內角計算

📖 簡報講義

🧪 互動探索：正 n 邊形角度計算

6
正六邊形：內角 = 120°、外角 = 60°
📝 形成性評量：英國新版 1 英鎊硬幣為正十二邊形，它的每一個內角為多少度？
正十二邊形的每一個外角 = 360° ÷ 12 = 30°，每一個內角 = 180° − 30° = 150°。達人捷徑：先算外角再用互補求內角！

📊常見錯誤分析

📊 教師｜學生常見錯誤

  • 誤以為「兩角必須在直角內」才算互餘（其實只要和為 90° 即可）
  • 誤以為「兩角必須在平角內」才算互補
  • 對頂角：誤以為共用頂點就是對頂角（需「兩直線交於一點」且不相鄰）
  • 外角定理：把「相鄰內角」和「不相鄰內角」搞混
  • 多邊形內角和：從內部一點分割時三角形數量計算錯誤
  • 「正」六邊形才是每個內角 120°，一般六邊形不一定
